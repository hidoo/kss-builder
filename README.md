## Demo

See [the example style guide](https://hidoo.github.io/kss-builder/) built with this builder.

## Installation

```sh
npm install --save-dev kss @hidoo/kss-builder
```

## Usage

```bash
kss path/to/css path/to/dest --builder @hidoo/kss-builder
```
